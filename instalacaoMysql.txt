Pro pessoal do Arch Linux


1. Instale o mysql

sudo pacman -S mysql


2. Verifique se foi instalado

    sudo mysqld --version

   

3. Depois de confirmar execute esse comando

    sudo mysql_install_db --user=mysql --basedir=/usr --datadir=/var/lib/mysql


4. Inicie o mysql:

        sudo systemctl start mysqld
        sudo systemctl status mysqld



5. Verifique o status do active estiver ativo e rodando:

    ● mariadb.service - MariaDB 10.9.3 database server
         Loaded: loaded (/usr/lib/systemd/system/mariadb.service; disabled; preset: disabled)
         Active: active (running) since Tue 2022-09-27 04:52:59 UTC; 6h ago


6. Rode o modo de instalação segura

     sudo mysql_secure_installation


7. Digite YES até aparecer isso, digite y e coloca a senha que usará no mysql

     You already have your root account protected, so you can safely answer 'n'.[y/n]



8. Digite YES para todas as configurações que vão ser feitas.


9. Para configurar o mysql, digite o comando abaixo

mysql -u root -p


Digite sua senha

Enter password: Sua senha


10. Dentro do mysql, crie o seu usuário:


    CREATE USER 'seu_usuario'@'localhost' IDENTIFIED BY 'sua_senha';



11. Garanta os privilégios de admin para seu usuário recém criado e saia da aplicação:


        GRANT ALL ON *.* TO 'seu_usuario'@'localhost' WITH GRANT OPTION;
        FLUSH PRIVILEGES;
        EXIT;

OBS: Toda vez que for usar o mySql e tenha ligado ou reinicado o computador, use o comando da etapa 4:
systemctl start mysqld
